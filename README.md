# VPN and VK Видео Autotests

## Технологии и инструменты
Appium, Java 21, Selenide, IntelliJ IDEA, Android Studio (Android 11 / api 30), Maven.

## Запуск
1. Установить Appium Server
2. Поднять эмулятор (Pixel_3_API_30)
3. Установить переменные окружения
4. Запустить:
   - кнопка "Play" у каждого теста (или их классов) через интерфейс 
   - либо через "mvn clean test";
